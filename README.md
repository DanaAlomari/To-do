# To-do

Work .It
Is a website to Plan, manage and track all your to-do's and checklists in one flexible Website.

Features :
Draggable Cards
Register & Login
Save users Todo's
Edit todo cards
sort by priority and status
Profile page
delete todo cards
Change theme
tracking logged users
